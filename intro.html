Download And Configuration:

Step One:

For startup download Git. For that goto: Download and download Git for your operating system

Step Two:

You need a github account to keep track of your Work. For that goto Github Signup to create your account. If you already have a account login to it.

Creating Repository

You need a repository to store your files.

Step One:

To create Repository Login to your account and hit the + Icon at the top right corner -> New Repository

Step Two:

Give your Repository a Name and then click on Create Repository

Step Three:

After you Repository is Created You will be able see this Page.

Add Git to your Project Folder

Step One:

Go to Your Project folder and in the address bar enter command prompt and press Enter

Step Two:

In command prompt enter "git init" and press enter. You will see a folder named ".git" created in your project folder.

Add Files to Local Repository

Step One:

To add Files to Local Repository use Command "git add filename" OR "git add ." to add all present files.

Step Two:

Then enter command "git status" to check status of your files and repositories

Green Files are the ones for synced with repository and red for that are unsynced.

Push Files to your Github repository

Go to Step "Craeting Repository" Part 3 and keep in view the block that States "â€¦or create a new repository on the command line"

Step One:

Commit you changes to your repository with command "git commit -m "name of your choice"

Step Two:

Select your branch with command "git branch -M main"

Step Three:

Connect to your Github Repository with command "git remote add origin YourGithubRepolink"

Step Four:

Push Your Files to Github Repository with command "git push -u origin main"

Step Five:

Now Refresh your Web Browser and you will see your Files in you Github Repository

Push new created and updated files

Step One:

To push new created files and updated files run Commands "git add .", "git commit -m "name of your choice", "git push -u origin main" one after the other

Step Two:

You can track all your commits from the Commits option
